# AI StyleMate

[cite_start]AI StyleMate is an AI-powered web service that provides personalized hairstyle and hair color recommendations by analyzing a user's face shape and personal skin tone[cite: 12]. [cite_start]It features a real-time AR try-on tool, allowing users to preview styles directly on their own faces to reduce uncertainty before visiting a salon[cite: 18, 19].
